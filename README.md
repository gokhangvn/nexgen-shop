# NexGen E-Ticaret

Gerçek zamanlı envanter yönetimi ve dağıtık sistem mimarisine sahip yüksek performanslı çok kanallı kontrol paneli.

## Proje Yapısı

- `frontend/` - React 19 uygulaması, Tailwind CSS ile.
- `backend/` - Node.js sunucusu (Express) ve Redis entegrasyonu.

## Başlangıç

Her iki klasörde de `npm install` veya tercih edilen paket yöneticisi ile bağımlılıkları yükleyin.

Frontend için:
```sh
cd frontend
npm run dev # veya yarn dev
```

Backend için:
```sh
cd backend
node index.js
```
